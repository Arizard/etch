A color palette manager which lets you name the colors.

I didn't like how coolors keeps asking me to upgrade for more shit that I don't need. In the spirit of anti-vibe-coding I made my own color palette manager for my specific purpose.

Paste a blob of text into the box below. It will extract the hex codes and put them into a palette which you can then modify. When you update colors they're reflected in the csv output below the palette.
